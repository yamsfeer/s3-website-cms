# UE 交互逻辑

## 前台访客流程

### 首页浏览

```
[访客访问 /] --> [加载首页]
                      |
                      v
              [展示轮播图] --> [自动播放/手动切换]
                      |
                      v
              [展示公司简介]
                      |
                      v
              [展示热门服务] --> [点击某服务] --> [/services/:id]
                      |
                      v
              [展示最新动态] --> [点击某新闻] --> [/news/:id]
```

### 服务浏览流程

```
[访客访问 /services] --> [加载服务列表]
                              |
                              v
                    [展示所有启用的服务卡片]
                              |
                              v
                    [点击某服务] --> [/services/:id]
                                        |
                                        v
                                  [展示服务详情]
                                        |
                                        v
                                  [点击"立即咨询"] --> [/contact]
```

### 新闻浏览流程

```
[访客访问 /news] --> [加载新闻列表]
                           |
                           v
                   [展示已发布新闻列表（封面+标题+日期）]
                           |
                           v
                   [点击某新闻] --> [/news/:id]
                                       |
                                       v
                                 [展示新闻详情]
                                       |
                                       v
                                 [点击"返回列表"] --> [/news]
```

### 在线留言流程

```
[访客访问 /contact] --> [显示留言表单+公司联系方式]
                               |
                               v
                       [填写姓名] --> [验证：不能为空]
                               |
                               v
                       [填写电话] --> [验证：不能为空，格式校验]
                               |
                       +--------+--------+
                       |                 |
                       v                 v
                   [格式正确]        [格式错误]
                       |                 |
                       v                 v
               [填写留言内容]     [显示错误提示]
                       |                 |
                       v                 |
               [验证：不能为空]          |
                       |                 |
               +-------+-------+        |
               |               |        |
               v               v        |
           [有效]          [无效]       |
               |               |        |
               v               v        |
        [点击提交按钮]   [显示错误提示]  |
               |                          |
               v                          |
        [提交到 POST /api/messages]       |
               |                          |
       +-------+-------+                  |
       |               |                  |
       v               v                  |
   [成功]           [失败]                |
       |               |                  |
       v               v                  |
 [显示成功提示]   [显示错误信息]          |
 [表单清空]       [保留已填数据]          |
```

## 后台管理流程

### 登录流程

```
[访问 /admin/*] --> [检查 LocalStorage Token]
                          |
                +---------+---------+
                |                   |
                v                   v
            [有 Token]          [无 Token]
                |                   |
                v                   v
        [验证 Token 有效性]   [跳转 /admin/login]
                |                           |
        +-------+-------+                   v
        |               |           [显示登录表单]
        v               v                   |
    [有效]           [无效]                 v
        |               |           [输入用户名+密码]
        v               v                   |
  [进入后台]     [跳转登录页]               v
                                    [点击登录]
                                         |
                                         v
                               [POST /api/auth/login]
                                         |
                               +---------+---------+
                               |                   |
                               v                   v
                           [成功]              [失败]
                               |                   |
                               v                   v
                       [存储 Token 到        [显示错误提示]
                        LocalStorage]         [清空密码框]
                               |
                               v
                       [跳转 /admin/dashboard]
```

### 留言管理流程

```
[访问 /admin/messages] --> [加载留言列表]
                                  |
                                  v
                      [展示留言列表（分页）]
                                  |
                                  v
                      [未处理留言高亮/角标]
                                  |
                +-----------------+-----------------+
                |                                   |
                v                                   v
          [查看留言详情]                      [标记已处理]
                |                                   |
                v                                   v
          [展开显示完整内容]                  [PUT /api/messages/:id/read]
                |                                   |
                v                                   v

                                                    |
                                                    v
                                            [角标数字-1]
```


## 交互状态

### Token 过期处理

```
[任意 API 请求] --> [返回 401]
                          |
                          v
                  [清除 LocalStorage Token]
                          |
                          v
                  [跳转 /admin/login]
                          |
                          v
                  [显示"登录已过期，请重新登录"]
```

```
[点击上传按钮] --> [弹出文件选择器]
                          |
                          v
                  [选择图片文件]
                          |
                  +-------+-------+
                  |               |
                  v               v
              [格式正确]     [格式不支持]
                  |               |
                  v               v
          [显示上传中状态] [显示"不支持的文件格式"]
                  |
          +-------+-------+
          |               |
          v               v

          |               |
          v               v
  [显示图片预览]   [显示"上传失败"]
  [返回图片URL]    [可重试]
```
